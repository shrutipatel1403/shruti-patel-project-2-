# Employee Attrition Analysis - Power BI Project

## Overview
This project analyzes employee attrition and workforce trends using Power BI.

## Tools Used
- Power BI
- DAX
- Excel/CSV

## Key Insights
- Overtime increases attrition risk
- Poor work-life balance leads to higher employee exits
- Early-tenure employees are more likely to leave

## Files Included
- Power BI Dashboard (.pbix)
- Dataset
- Project Report
